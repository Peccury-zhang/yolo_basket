# 任务：basket 掩膜底部 1/4 提取 + 上边缘线段拟合采样（v5 最终方案）

## 1. 任务目标

1. 使用 YOLO11-Seg 模型对输入图像进行 basket 实例分割。
2. 提取掩膜的**底部 1/4 区域**（红色标注）。
3. 在底部 1/4 区域的**上边缘拟合一条直线段**（黄色），并在线段上**均匀采样 10 个点**（蓝色圆点）。
4. 保存标注图像和采样点坐标。

## 2. 历史方案演进

| 版本 | 方案 | 问题 |
|------|------|------|
| v1 | 全局水平切割 `y_cut` | basket 倾斜时一侧超 1/4，另一侧不足 |
| v2 | 逐列比例裁剪 | 透视导致切割线呈梯形，不与 basket 边缘平行 |
| v3 | 旋转矩形 75% 插值 + bitwise_and | 切割区域正确 |
| v4 | 直接用 CL/CR 或最左最右上边缘点画线段 | 旋转矩形比掩膜宽导致线段超出；最左最右点在窄尖角底部导致线段偏低 |
| **v5** | **fitLine 拟合上边缘 + 裁掉两端 5% 尖角** | **最终方案** |

## 3. v5 最终方案

### 3.1 底部 1/4 区域提取（沿用 v3）

1. 用 `cv2.minAreaRect` 对掩膜求**最小面积旋转矩形**，`cv2.boxPoints` 获取四角点。
2. 四角点按 y 排序分为顶部（TL, TR）和底部（BL, BR），再按 x 区分左右。
3. 在 75% 处线性插值得到切割线端点：
   - $CL = TL + 0.75 \times (BL - TL)$
   - $CR = TR + 0.75 \times (BR - TR)$
4. 由 CL, CR, BR, BL 构成四边形，与原始掩膜 `bitwise_and` 取交集，得到底部 1/4 区域。

```
    TL ─────────── TR          ← 旋转矩形顶边
   ╱                 ╲
  ╱                   ╲
 CL ─────────── CR            ← 75% 插值线
╱   ███ 红色区域 ███   ╲
BL ─────────── BR              ← 旋转矩形底边
```

### 3.2 上边缘线段拟合 + 采样（v5 核心改进）

**问题**：底部 1/4 区域（`bottom_mask`）是旋转矩形四边形与原始掩膜的交集，两端会形成窄尖角。直接取最左/最右列的上边缘点会落在尖角底部，导致线段偏低或超出。

**解决方案**：

1. **逐列扫描上边缘**：对 `bottom_mask` 的每一列，找到最小 y 值，收集所有 `(x, y_min)` 上边缘点。
2. **`cv2.fitLine` 拟合直线**：对全部上边缘点拟合直线，得到方向向量 `(vx, vy)` 和线上一点 `(cx, cy)`。
3. **裁掉两端 5% 尖角**：取 x 的第 5 百分位到第 95 百分位范围，避免两端窄尖角影响端点位置。
4. **计算线段端点**：在裁掉尖角后的 x 范围内，用拟合直线计算 y 坐标。
5. **均匀采样 10 个点**：$P_i = P_{left} + \frac{i}{9} \times (P_{right} - P_{left})$，$i = 0, 1, ..., 9$

```
  ╱                   ╲
 CL                     CR     ← 旋转矩形切割线（可能超出掩膜）
╱  L● ● ● ● ● ● ● ● ●R ╲    ← 拟合线段（裁掉 5% 尖角，贴合掩膜）
╱   ████ 红色区域 ████   ╲
BL ─────────── BR

L = fitLine 在 x_5% 处的点
R = fitLine 在 x_95% 处的点
```

## 4. 效果说明

| 元素 | 颜色 | 说明 |
|------|------|------|
| 完整 basket 掩膜 | 绿色半透明 | 模型分割输出的完整掩膜 |
| 底部 1/4 掩膜 | 红色半透明 | 旋转矩形切割与掩膜的交集 |
| 上边缘拟合线段 | 黄色线 | fitLine 拟合，裁掉两端 5% 尖角 |
| 均匀采样点 | 蓝色圆点 | 线段上等间距 10 个点 |

## 5. 输入输出

### 输入
- `source_img/{name}.png`：待预测的图像（支持 png/jpg/jpeg/bmp/tif）
- `best.pt`：YOLO11-Seg 训练好的模型权重

### 输出
- `output_img/{name}_mask.png`：带完整掩膜 + 底部 1/4 掩膜 + 拟合线段 + 采样点的标注图像
- `output_img/{name}_mask.txt`：掩膜坐标文件（完整掩膜 + 底部 1/4 掩膜轮廓坐标）
- `output_img/{name}_points.txt`：10 个采样点坐标，每行格式 `x,y`

## 6. 关键技术点

- **`cv2.minAreaRect` + `cv2.boxPoints`**：获取旋转矩形角点，提供 basket 倾斜方向。
- **75% 线性插值**：沿旋转矩形短边在 3/4 处生成与 basket 平行的切割线。
- **`cv2.bitwise_and`**：切割四边形与原始掩膜取交集，确保底部 1/4 不超出掩膜。
- **`cv2.fitLine`**：对上边缘点集拟合直线，抗噪声，方向准确。
- **百分位裁剪（5%~95%）**：避免交集区域两端窄尖角对线段端点的影响。
- **等间距向量插值**：在拟合线段上均匀生成 10 个采样点。
- 支持批量处理 `source_img/` 下所有图片，多 basket 实例逐一处理。
